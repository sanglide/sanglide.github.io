Data Structure + Algorithm

1. 表驱动

csdn解说

对于依据关键字选择执行代码switch-case与if-else语句块，当选择较多时维护性变差。选择表驱动来优化。

对于switch-case与if-else语句块，有三个变化点：

  1. 将关键字与处理代码关联；

  2. 以关键字选择分支，执行处理代码；

  3. 实现关键字的处理代码；

在表驱动中：

  1. 将变化点2做成一个[关键字:处理函数]的映射结构，在独立函数中赋值

  2. 将变化点三，做成一个查找关键字，执行对应函数的简单函数。（更加抽象，易于维护）

ppt中的例子将结构做成一个结构体，实际中也可以用map；ppt中还使用一个enum类型将数字与文字对应，使代码可读性更高。
